Tentang Pemrograman

Pemrograman adalah proses menulis instruksi atau kode untuk memberi tahu komputer cara menjalankan tugas-tugas tertentu. Bahasa pemrograman memungkinkan pengembang untuk berkomunikasi dengan komputer dalam bentuk yang dapat dipahami oleh mesin. Dengan pemrograman, kita bisa menciptakan berbagai aplikasi, situs web, sistem operasi, dan perangkat lunak lainnya.

Bahasa apa saja yang dipakai?

Ada banyak bahasa pemrograman, masing-masing dengan tujuan dan kelebihannya sendiri, seperti: HTML, CSS, dan JavaScript: Kombinasi ini banyak digunakan untuk pengembangan web. HTML mengatur struktur halaman web, CSS mengatur tampilan, dan JavaScript menambah interaktivitas. Python: Bahasa serbaguna yang sering digunakan untuk pengembangan aplikasi, analisis data, kecerdasan buatan, dan pembelajaran mesin. Java: Populer untuk pengembangan aplikasi Android, juga digunakan dalam pengembangan aplikasi yang skalanya besar. C dan C++: Dipakai dalam pemrograman sistem, perangkat lunak yang membutuhkan performa tinggi, serta pengembangan game. SQL: Bahasa yang digunakan untuk manajemen basis data, memungkinkan kita untuk menyimpan, mengelola, dan mengambil data."

Belajar HTML

1. Menulis "Hello, World!"

              
    <!DOCTYPE html>
    <html>
      <body>
        <h1>Hello, World!</h1>
      </body>
    </html>

2. Membuat Tabel

              
    <table border="1">
      <tr>
        <th>Nama</th>
        <th>Umur</th>
      </tr>
      <tr>
        <td>Dimas</td>
        <td>20</td>
      </tr>
    </table>

3. Penggunaan List dan Navbar

              
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
    
    <nav>
      <a href="#">Home</a> | <a href="#">About</a> | <a href="#">Contact</a>
    </nav>

Belajar CSS

1. Mengubah Warna Teks

              
    <!DOCTYPE html>
    <html>
      <head>
        <style>
          h1 {
            color: blue;
          }
        </style>
      </head>
      <body>
        <h1>Teks Berwarna Biru</h1>
      </body>
    </html>

2. Menambahkan Border

              
    <!DOCTYPE html>
    <html>
      <head>
        <style>
          div {
            border: 2px solid black;
            padding: 10px;
          }
        </style>
      </head>
      <body>
        <div>Ini adalah kotak dengan border</div>
      </body>
    </html>

3. Mengubah Background

              
    <!DOCTYPE html>
    <html>
      <head>
        <style>
          body {
            background-color: lightgray;
          }
        </style>
      </head>
      <body>
        <h1>Halaman dengan Background Abu-Abu</h1>
      </body>
    </html>

Belajar JavaScript

1. Menampilkan Pesan dengan Alert

              
    <!DOCTYPE html>
    <html>
      <head>
        <title>Contoh Alert</title>
        <script>
          function showMessage() {
            alert('Halo, ini pesan dari JavaScript!');
          }
        </script>
      </head>
      <body>
        <button onclick="showMessage()">Klik Saya</button>
      </body>
    </html>

2. Menampilkan Teks di Halaman

              
    <!DOCTYPE html>
    <html>
      <head>
        <title>Contoh Menampilkan Teks</title>
        <script>
          function displayText() {
            document.getElementById('output').innerText = 'Halo, ini teks dari JavaScript!';
          }
        </script>
      </head>
      <body>
        <button onclick="displayText()">Tampilkan Teks</button>
        <div id="output" style="margin-top: 10px;"></div>
      </body>
    </html>

3. Perhitungan Sederhana

              
    <!DOCTYPE html>
    <html>
      <head>
        <title>Contoh Perhitungan</title>
        <script>
          function calculateSum() {
            var a = 5;
            var b = 3;
            var sum = a + b;
            document.getElementById('result').innerText = 'Hasil: ' + sum;
          }
        </script>
      </head>
      <body>
        <button onclick="calculateSum()">Hitung 5 + 3</button>
        <div id="result" style="margin-top: 10px;"></div>
      </body>
    </html>
